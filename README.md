# 使用方法
在终端窗口下，定位到Hexo站点目录下。
```bash
cd your-hexo-site
git clone https://github.com/kmac007/next-mist-kmac007.git themes/next
```
在`站点`的配置文件`_config.yml`下将`theme`改为：
```
theme: next
```
`主题`配置文件为中:
scheme默认为Mist，无需修改。

在站点目录下执行`hexo s`即可看到预览
```bash
hexo s
```
